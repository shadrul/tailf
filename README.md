# TAILF SERVER

A FLask server to send only the updated data of a file placed on server, to client using websockets.


## Tech

FLask, Websockets


## Usage

Create a virtulal env like
```sh
virtualenv <virtualenv_name>
```

Download all the required libs by following comamnd
```sh
pip3 install -r /path/to/requirements.txt
```

Now, run your sever by moving to the workig directory(interview1)
```sh
python3 app.py
```
Now, in your browser head over to "http://127.0.0.1:5000/" and see the magic.
